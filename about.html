How long have you been practicing machine learning?

I discovered machine learning at the end of 2018 after a friend of mine introduced it's gist to me with a simple project of his. Since then, I have immersed myself in data science communities on Medium and across the web.

I have experience in building machine learning solutions for NLP, Computer Vision and Data Modelling tasks. I have built models to assist in sentiment analysis, demographic clustering and topic modelling.

Why are you interested in Machine Learning/ Data Science?

I am curious by nature. Machine learning and data science are both excellent tools to understand the world better.

Despite the hype over integrating AI into existing technologies, the implementation of machine learning (especially NLP and CV) is still in its infancy. This field is the wildwest, and I want to work with the pioneers and innovators that create new and amazing things.

What kind of work do you see yourself doing in coming years?

I am interested in building modular machine learning applications that can communicate with existing production systems.

Position titles that do this work are not well defined. I describe my perfect future position to be a mix of data science, machine learning engineering, and devops. Essentially a machine learning specialist with the skills to deploy a model to production.
